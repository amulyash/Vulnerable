This repository contains snippets of intentionally vulnerable code that can help learners understand 
and identify common security issues such as Cross-Site Scripting (XSS), SQL Injection, Path Traversal, Insecure Direct Object References (IDOR), and more.

The code is provided purely for educational purposes and is not intended to function as a complete or 
runnable application. It is designed for individuals interested in learning about insecure coding patterns, vulnerability identification, and secure code review practices.



Enjoy Hunting :)
